# 기술 의사결정 기록 (ADR)

## ADR-001: Google Apps Script 플랫폼 선택

**날짜**: 2026-01-19  
**상태**: ✅ 승인됨  
**결정자**: 프로젝트 팀

### 컨텍스트
블로그 자동화 시스템을 구축하기 위해 플랫폼을 선택해야 했습니다.

### 결정
Google Apps Script를 메인 플랫폼으로 선택

### 근거
- **통합성**: Google Drive, Docs, Sheets와 네이티브 통합
- **무료**: 별도 서버 비용 없음
- **접근성**: 웹 브라우저에서 바로 실행 가능
- **자동화**: 트리거 및 스케줄링 기능 내장

### 결과
- 서버 관리 불필요
- Google 생태계 내에서 완전한 자동화 구현
- 파일 관리 및 권한 제어 간소화

## ADR-002: Claude AI (Anthropic) 선택

**날짜**: 2026-01-19  
**상태**: ✅ 승인됨  
**결정자**: 프로젝트 팀

### 컨텍스트
SEO 콘텐츠 생성을 위한 AI 모델 선택이 필요했습니다.

### 결정
Claude Sonnet 4.5 (Anthropic API) 사용

### 근거
- **긴 컨텍스트**: 대용량 HTML/TXT 파일 처리 가능
- **품질**: 자연스러운 한국어 글쓰기
- **프롬프트 엔지니어링**: System/User 메시지 분리로 정교한 제어
- **안정성**: 일관된 출력 형식

### 대안
- OpenAI GPT-4: 비용이 높고 컨텍스트 제한
- Perplexity AI: 검색 특화, 글쓰기 품질 낮음

### 결과
- 고품질 SEO 콘텐츠 생성
- 템플릿 기반 스타일 제어 가능
- API 속도 제한 관리 필요 (5분당 1개)

## ADR-003: 메모리 기반 처리 방식

**날짜**: 2026-01-19  
**상태**: ✅ 승인됨  
**결정자**: 프로젝트 팀

### 컨텍스트
중간 JSON 파일 저장이 Google Drive 용량을 차지하고 처리 속도를 저하시켰습니다.

### 결정
`runCompleteProcessInMemory()` 함수로 메모리에서만 처리

### 근거
- **속도**: 파일 I/O 최소화
- **용량**: Drive 저장 공간 절약
- **단순성**: 중간 파일 관리 불필요

### 트레이드오프
- 에러 발생 시 중간 데이터 복구 불가
- 디버깅 어려움 증가

### 결과
- 처리 시간 약 30% 단축
- Drive 용량 사용량 50% 감소
- 에러 로깅 강화 필요성 대두

## ADR-004: 스타일 파라미터 시스템

**날짜**: 2026-01-19  
**상태**: ✅ 승인됨  
**결정자**: 프로젝트 팀

### 컨텍스트
다양한 글쓰기 톤과 스타일을 지원해야 했습니다.

### 결정
Google Sheets 기반 7가지 스타일 파라미터 시스템 구축

### 파라미터
1. 글톤 (writing_tone)
2. 문장스타일 (sentence_style)
3. 개인터치 (personal_touch)
4. 시각풍부도 (visual_richness)
5. 내러티브플로우 (narrative_flow)
6. 전문성레벨 (expertise_level)
7. 평균문장길이 (avg_sentence_len)

### 근거
- **유연성**: 코드 수정 없이 스타일 조정
- **확장성**: 새로운 스타일 쉽게 추가
- **사용성**: 비개발자도 스프레드시트에서 관리 가능

### 결과
- 5가지 기본 스타일 프리셋 제공
- 프롬프트 자동 생성으로 일관성 유지
- 사용자 커스터마이징 용이

## ADR-005: Apple + 네이버 하이브리드 디자인

**날짜**: 2026-01-19  
**상태**: ✅ 승인됨  
**결정자**: 프로젝트 팀

### 컨텍스트
Google Docs 출력물의 디자인 방향성 결정이 필요했습니다.

### 결정
Apple의 미니멀리즘 + 네이버 블로그의 친근함 결합

### 근거
- **Apple 스타일**: 전문성, 가독성, 깔끔함
  - 색상: #1d1d1f (다크 그레이), #86868b (라이트 그레이)
  - 폰트: Noto Sans
  - 간격: 1.8 line-spacing
- **네이버 블로그 스타일**: 친근함, 접근성
  - 이모지 하이라이트 (💰, 💡, 📊)
  - 요약 박스
  - 친근한 마무리 멘트

### 결과
- 전문성과 친근함의 균형
- 높은 가독성
- SEO 친화적 구조
