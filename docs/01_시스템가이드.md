# 01_시스템가이드

## 권장 실행 순서
1. 입력 폴더에 원본 파일 업로드
2. 시트1 설정값 입력
3. Apps Script에서 `runV7HTMLPipeline()` 실행
4. JSON 출력 폴더에서 중간 산출물 확인
5. 결과 출력 폴더에서 최종 HTML 확인

## 시트1 입력 방법

### A2 강조키워드
- 글에서 반드시 살아야 하는 핵심 키워드
- 쉼표로 구분
- 예: `방수석고, 친환경자재, 시공주의사항`

### B2 스타일 번호
- 시트2에서 불러올 스타일 번호
- 숫자만 입력
- 값이 비어 있거나 잘못되면 기본값 `1` 사용

### C2 검색최적화키워드
- SEO 노출용 키워드
- 쉼표로 구분
- 예: `석고보드, 방수석고보드, 석고보드 시공`

### D2 템플릿
- 시트3 템플릿명과 정확히 일치해야 함
- 예: `제품소개형`, `시공가이드형`, `트렌드분석형`

## 시트1 운영값
- E열: 최신반영여부
- F열: 발행모드
  - 현재 행 선택 판단에 사용되며 `자동`, `수동승인` 중 하나여야 함
- G열: 상태
- H열: 발행시간
- I열: URL
- J열: 이미지소스
  - `자동생성`, `직접업로드`

## 시트1 자동 출력 열
- M열: `image1_url`
- O열: `업로드폴더URL`
- P열: `필요사진수`
- Q열: `파일명가이드`
- R열: `image2_svg_url`

## 메뉴 버튼
- `📸 자동생성 모드로 설정`
- `📁 직접업로드 모드로 설정`
- `🚀 전체실행 — 자동생성 (글+이미지+발행)`
- `① 글 생성`
- `② 이미지 생성 (OpenAI)`
- `② 사진 업로드 폴더 열기`
- `③ 발행`
- `🔄 초기화 (새 글 시작)`

## 현재 권장 버튼/함수
- 권장: Apps Script에서 `runV7HTMLPipeline()`
- 보조:
  - `setModeAuto_()`
  - `setModeManual_()`
  - `resetForNewPost()`

## 폴더 구조
- 입력 폴더
- 처리완료 원본 폴더
- JSON 출력 폴더
- 결과 출력 폴더

## 운영 팁
- 한 번에 1개 파일 기준으로 운영하는 편이 가장 안전하다
- 실행 전에 시트1 A~D, F, J가 올바른지 먼저 확인한다
- 결과가 이상하면 JSON 출력 폴더의 `_final_seo.json`부터 확인한다
