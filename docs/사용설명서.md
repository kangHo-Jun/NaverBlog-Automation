# 시트1 A~H열 사용설명서

본 문서는 `NaverBlog-Automation` 시스템의 **시트1(Main Sheet)** A열부터 H열까지의 역할, 입력 형식, 시스템 내부 연동 및 영향받는 기능을 설명합니다.

---

## 📊 시트1 A~H열 상세 명세

| 열 | 이름 | 입력값 예시 | 관련 내부 함수 / 시스템 동작 | 상세 설명 및 영향도 |
| :--- | :--- | :--- | :--- | :--- |
| **A열** | **강조 키워드** | `방수석고, 합판 규격, 단열재 시공` | `getHighlightKeywordsFromA2()` | 본문 작성 시 반드시 강조(Bold 등)하여 노출할 키워드들입니다. 쉼표(`,`)로 구분하여 최대 5개까지 입력할 수 있습니다. AI 프롬프트 생성 시 핵심 키워드로 포함됩니다. |
| **B열** | **스타일 번호** | `1` 또는 `2` | `styleNumber = mainSheet.getRange('B2').getValue()` | 글톤, 문장 스타일, 시각적 풍부도 등 글쓰기 톤앤매너 스타일 번호입니다. **시트2(Style)**의 특정 번호 레코드와 맵핑되어 AI 콘텐츠 생성 프롬프트에 스타일 가이드라인을 제공합니다. |
| **C열** | **SEO 키워드** | `석고보드 규격, 친환경 단열재` | `getSEOKeywordsFromC2()` | 검색 엔진 최적화(SEO)를 위한 핵심 타겟 키워드 목록입니다. 쉼표(`,`)로 구분하여 입력하며, 본문 전처리 및 최종 SEO HTML/Docs 빌드 파이프라인에서 키워드 강제 노출 및 태그 배치에 핵심적인 역할을 합니다. |
| **D열** | **템플릿 선택** | `유튜브스크립트형`, `제품소개형` | `selectedTemplateName = mainSheet.getRange('D2').getValue()` | 생성할 글의 레이아웃 구조를 결정하는 템플릿명입니다. **시트3(Template)**의 템플릿 레이아웃과 맵핑되어 최종 발행될 HTML의 구조적 프레임워크를 지정합니다. |
| **E열** | **Gemini 검색 여부** | `Y` 또는 `N` | `getSearchOptionFromI2()` (E2 셀에서 읽음) | Gemini를 통한 최신 트렌드/웹 검색 데이터 분석 활용 여부입니다. `Y` 설정 시 실시간 검색 기반의 정보가 수집되어 글에 반영되며, 미입력 시 기본값은 `Y`로 동작합니다. |
| **F열** | **발행 모드** | `자동` 또는 `수동승인` | `getPublishControlRow_()` | 해당 행의 프로세스 처리 대상 여부를 판별하는 필터링 조건입니다. 이 열이 `'자동'` 또는 `'수동승인'`으로 되어 있어야 배포 파이프라인 대상 행으로 선택됩니다. |
| **G열** | **상태** | `발행중`, `완료`, `오류`, `이미지재생성` | `updatePublishStatus_()` | 시스템 실행 상태가 실시간으로 자동 기입되는 영역입니다. 작업이 시작되면 `'발행중'`, 성공 시 `'완료'` 또는 오류 발생 시 `'오류'`가 업데이트됩니다. |
| **H열** | **예약 발행 시간** | `2026-06-25 10:00:00` | `controlRow.values[7]` | 블로그 포스트의 예약 발행 시점을 지정합니다. 날짜/시간 형식으로 기입 시 ISO 8601 포맷으로 변환되어 Blogger 예약 발행 API로 전달되며, 비어있을 시 즉시 발행됩니다. |

---

## ⚙️ 시스템 동작 흐름 및 연동 규칙

1. **대상 행 스캔 (`getPublishControlRow_`)**
   - **F열(발행 모드)**이 `'자동'` 또는 `'수동승인'`이고, **I열(발행 URL)**이 비어있는 행을 대상으로 최상단 행부터 1개씩 순차적으로 검색합니다.
   
2. **콘텐츠 스타일링 매핑**
   - **B열(스타일 번호)**의 값을 기준으로 시트2에서 글톤(Tone), 문장 스타일(Sentence Style), 평균 문장 길이(Avg Sentence Length) 등의 세부 규칙을 로드합니다.
   - **D열(템플릿 선택)**의 값을 기준으로 시트3에서 해당 템플릿 레이아웃 구성을 가져옵니다.

3. **초기화 동작 (`resetForNewPost`)**
   - 새로운 글 생성을 위해 초기화 함수를 실행하면 **G열(상태)**, **H열(예약 발행 시간)**, **I열(발행 URL)** 등 주요 실행 이력 열들의 데이터가 자동으로 지워집니다.
