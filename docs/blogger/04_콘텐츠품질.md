# 04_콘텐츠품질

## 가격 표현 규칙
- 제공 자료에 없는 원 단위 가격 생성 금지
- 가격 비교는 `상대적으로 낮음`, `중간`, `높음`처럼 상대 표현 우선
- `초기 비용은 높을 수 있음` 같은 조건부 문장은 허용
- 퍼센트 차이는 제공 자료에 명시된 경우에만 사용

## 용어 현지화 규칙
- 영어 참고글 직역 금지
- 국내 건축자재 현장 용어로 변환
- 약어 `LVB`, `LVL`, `MDF`, `OSB`, `KS`, `E0`, `T`, `mm`, `MPa`는 유지

## 비교표 출력 규칙
- 두 개 이상 자재, 성능, 용도 비교 시 반드시 Markdown 테이블 출력
- 헤더 행, 구분선 행, 본문 행 모두 포함
- 테이블 앞뒤 불필요한 빈 줄 금지
- 최소 4개 이상 비교 항목
- 제공 자료 없는 수치 임의 생성 금지
- 수치 근거가 없으면 `상대적으로 낮음`, `상대적으로 높음`, `조건 확인 필요`로 표현

## SEO 키워드 자연어 규칙
- 지역명+자재명 키워드는 본문에서 자연어 문장으로 풀어쓴다
- 예: `안양 석고보드` 대신 `안양에서 석고보드를 찾는 경우`
- `title`, `meta_description`, `tags`는 원문 키워드 사용 가능
- 본문은 자연어 변형 우선
- SEO 키워드를 고유명사처럼 반복하지 않는다

## 법규/기준 표현 규칙
- 법규 확정 표현 금지
- 인증, 방화, KS 관련 내용은 참고 기준으로만 설명
- 구조 계산이나 현장 적용 가능 여부를 단정하지 않는다
- 기준규정설명 유형은 특히 `조건 확인 필요`, `현장 확인 필요` 문장을 남긴다

## 자동 검수 규칙
- `fact_safety_check.contains_exact_price`
- `fact_safety_check.contains_unsourced_percent`
- `fact_safety_check.contains_unsourced_lifespan`
- `fact_safety_check.contains_awkward_seo_phrase`
- `fact_safety_check.comparison_sections_use_tables`

## 검수 해석
- `contains_exact_price=true`: 근거 없는 가격 노출 가능성 검토
- `contains_unsourced_percent=true`: 출처 없는 비율 수치 검토
- `contains_unsourced_lifespan=true`: 출처 없는 수명 표현 검토
- `contains_awkward_seo_phrase=true`: 키워드가 부자연스럽게 들어간 문장 검토
- `comparison_sections_use_tables=false`: 비교 구간에 Markdown 테이블이 빠졌는지 점검
