# Blogger 자동화 시스템 사용자 가이드

> 대상: 코딩 없이 이 시스템을 사용하는 초보 사용자

---

## 이 시스템이 하는 일

소스 파일(텍스트 또는 HTML)을 업로드하면 AI가 SEO 최적화 블로그 글을 자동으로 작성하고 Blogger에 발행합니다.

---

## 전체 실행 흐름

```
① 소스파일 업로드
   └ Drive 입력폴더에 .txt 또는 .html 파일 업로드

② runGenerateOnly() 실행
   └ AI가 SEO 최적화 글 자동 작성
   └ J열에 이미지 폴더 자동 생성
   └ K열에 필요한 사진 목록 자동 기록

③ 사진 업로드
   └ K열 이미지목록 확인
   └ J열 폴더에 01.jpg, 02.jpg ... 형식으로 업로드

④ runPublishOnly() 실행
   └ [자동]     → 즉시 Blogger 공개 발행
   └ [수동승인] → H열 체크 안 됨 → 승인대기 후 종료
                → H열 체크 후 재실행 → Blogger 임시저장(DRAFT) 발행
```

---

## 컨트롤 시트 입력 가이드 (시트1)

### 사용자가 직접 입력하는 열

| 열 | 항목 | 입력 예시 | 설명 |
|----|------|----------|------|
| A | 강조키워드 | 단 7mm의 차이가 만드는 압도적 개방감! | 글에서 강조할 핵심 문구 |
| B | 스타일번호 | 2 | 1~4 중 선택 (시트2 참고) |
| C | SEO키워드 | 안양창호, 안양 LX창호, 고급창호 | 쉼표로 구분 |
| D | 템플릿명 | 문제해결형 | 시트3 참고 |
| E | 최신반영여부 | y | y 또는 n |
| F | 발행모드 | 자동 | `자동` 또는 `수동승인` |
| H | 승인 | ☑ | 수동승인 모드일 때만 체크 |
| I | 발행시간 | 2026-05-10 10:00 | 비워두면 즉시 발행 |

### 시스템이 자동으로 기록하는 열 (절대 수정 금지)

| 열 | 항목 | 설명 |
|----|------|------|
| G | 상태 | 시스템이 자동 기록 |
| J | 이미지폴더 | runGenerateOnly() 실행 시 자동 생성 |
| K | 이미지목록 | 필요한 사진 목록 자동 기록 |
| L | Blogger URL | 발행 완료 후 자동 기록 |

---

## 발행모드 상세 설명

### 자동 모드
- F열에 `자동` 입력
- `runPublishOnly()` 실행하면 **즉시 Blogger에 공개 발행**

### 수동승인 모드
- F열에 `수동승인` 입력
- `runPublishOnly()` 실행 → H열 체크 안 되어 있으면 → G열 `승인대기` 기록 후 종료
- 글 내용 확인 후 H열 체크박스 **체크**
- `runPublishOnly()` 다시 실행 → Blogger **임시저장(DRAFT)** 으로 발행
- Blogger 편집기에서 직접 확인 후 공개 버튼 클릭

---

## 상태(G열) 종류

| 상태 | 의미 |
|------|------|
| 대기중 | 초기값 (사용자가 직접 입력) |
| 승인대기 | 수동승인 모드인데 H열 체크 안 됨 |
| 발행완료 | 발행 성공 |

---

## 사진 업로드 규칙

- K열 이미지목록을 보고 필요한 사진 종류 확인
- J열에 표시된 Drive 폴더에 업로드
- **파일명은 반드시 `01.jpg`, `02.jpg`, `03.jpg` 순서대로**
- 사진 없이도 발행 가능 (사진 없이 텍스트만 발행됨)

---

## 예약 발행 방법

- I열에 발행 원하는 시간 입력: `2026-05-10 10:00`
- 비워두면 즉시 발행
- `runPublishOnly()` 실행 시 해당 시간에 발행 예약됨

---

## 주의사항

- G, J, K, L열은 **절대 수정 금지** (시스템 자동 기록)
- 이미지 파일명은 반드시 `01.jpg` 형식 준수
- `runGenerateOnly()` 완료 후 반드시 사진 업로드 후 `runPublishOnly()` 실행
- 체크박스 체크 후 `runPublishOnly()` 를 **직접 실행**해야 발행됨 (자동 감지 안 됨)

## FAQ

**Q. 체크박스 체크했는데 왜 발행이 안 되나요?**
체크 후 Apps Script 편집기에서 `runPublishOnly()` 를 직접 실행해야 합니다.

**Q. 사진이 없어도 발행되나요?**
됩니다. 사진 없이 텍스트만 발행됩니다.

**Q. 예약발행은 어떻게 하나요?**
I열에 `2026-05-10 10:00` 형식으로 입력 후 `runPublishOnly()` 실행하면 됩니다.

**Q. 수동승인으로 발행하면 블로그에 바로 보이나요?**
아닙니다. Blogger 임시저장(DRAFT) 상태로 저장됩니다. Blogger 편집기에서 직접 공개 버튼을 눌러야 합니다.

**Q. runGenerateOnly()와 runPublishOnly()는 언제 실행하나요?**
- `runGenerateOnly()` → 글 작성 단계 (소스파일 업로드 후)
- `runPublishOnly()` → 발행 단계 (사진 업로드 후)
