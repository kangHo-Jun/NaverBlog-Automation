# Blogger 자동화 개발 체계

> 작성일: 2026-04-28 / 최종 업데이트: 2026-05-08 (img 스타일, 사진가이드 텍스트 제거, 중복발행 방지)

---

## 목표

> 소스파일(HTML/TXT) → Claude AI SEO 글 생성 → Drive 사진 매핑 → Blogger 자동 발행

---

## 전체 파이프라인

```
소스파일 (HTML / TXT)
    ↓
[전처리] Gemini 1.5 Flash
  - 컨트롤 시트(시트1) 설정값 기반 분석
  - summary / outline / key_facts / seo_addition 추출
  - E2 셀 값에 따라 Google 검색 그라운딩 ON/OFF
    ↓
[SEO 글 생성] Claude API (claude-sonnet-4-5-20250929)
  - 시트1 설정값 반영 (키워드, 스타일, 템플릿)
  - 한국어 SEO 최적화 블로그 글 생성
  - [사진1], [사진2] ... 플레이스홀더 자동 생성

── runGenerateOnly() ──────────────────────────────
  - Drive 폴더 자동 생성 (YYYYMMDD_글제목) → K열 기록
  - 사진 가이드 생성 → G열 기록 / 상태 G열 업데이트
    ↓
사용자: Drive 폴더에 사진 업로드 (수동)
  예) 📁 20260428_글제목/  →  01.jpg, 02.jpg ...

── runPublishOnly() ───────────────────────────────
  - L열 URL 존재 시 즉시 중단 (중복 발행 방지)
  - mapPhotosToPlaceholders()
      → [사진N] → 실제 이미지 URL 교체
      → 남은 [사진 N: ...] 가이드 텍스트 전체 제거
  - convertToBloggerHTML()
      → Blogger용 HTML 변환
      → img 태그: width:100%, max-width:800px, height:auto, display:block, margin:1em auto
  - publishToBlogger()         → Blogger API v3 발행
      F열 자동         → LIVE (즉시 공개)
      F열 수동승인     → H열 체크 시 DRAFT (임시저장)
                          H열 미체크 시 승인대기 보류
      I열 시간 입력    → ISO 8601 변환 후 예약발행
  - updateControlSheetAfterPublish() → L열 URL 기록
  - 발행 성공 후 _final_seo.json 자동 삭제
```

## 컨트롤 시트(시트1) 구조

### 설정 셀 (2행 고정)

| 셀 | 항목 | 설명 |
|----|------|------|
| A2 | 강조 키워드 | 최대 5개, 쉼표 구분 |
| B2 | 스타일 번호 | 1~5 (시트2 참조) |
| C2 | SEO 키워드 | 쉼표 구분 |
| D2 | 템플릿 선택 | 시트3 참조 (예: 제품소개형, 유튜브스크립트형) |
| E2 | 최신반영 Y/N | Y = Gemini Google 검색 그라운딩 ON |

### 발행 관리 열 (4행부터 데이터)

| 열 | 항목 | 내용 |
|----|------|------|
| F열 | 발행모드 | `자동` / `수동승인` |
| G열 | 상태 | `대기중` / `승인대기` / `발행완료` |
| H열 | 승인 | ☐ 체크박스 (수동승인 시 체크 필요) |
| I열 | 예약시간 | 비어있으면 즉시, 날짜시간 입력 시 예약발행 |
| J열 | (예비) | - |
| K열 | 이미지폴더 | Drive 폴더 URL 자동 입력 |
| L열 | Blogger URL | 발행 후 자동 입력 |
