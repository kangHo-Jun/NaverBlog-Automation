# 콘텐츠 품질 규칙

## 가격 표현 규칙

- 제공 자료에 없는 원 단위 가격 생성 금지
- 가격 비교는 `상대적으로 낮음/중간/높음`, `초기 비용은 높을 수 있음`처럼 상대 표현 사용
- % 차이는 제공 자료에 명시된 경우에만 사용

## 용어 현지화 규칙

- 영어 참고글 직역 금지
- 국내 건축자재 유통/시공 현장 용어로 변환
- `LVB`, `LVL`, `MDF`, `OSB`, `KS`, `E0`, `T`, `mm`, `MPa` 약어는 그대로 유지

## 비교표 출력 규칙

- 두 개 이상 자재/성능/용도 비교 시 반드시 Markdown 테이블 출력
- 헤더 행, 구분선 행, 본문 행 모두 포함
- 테이블 앞뒤 불필요한 빈 줄 금지
- 최소 4개 이상 비교 항목 유지
- 제공 자료 없는 수치 임의 생성 금지
- 수치 근거 없으면 `상대적으로 낮음`, `상대적으로 높음`, `조건 확인 필요`처럼 표현

## SEO 키워드 자연어 규칙

- 지역명+자재명 키워드는 본문에서 자연어로 풀어쓴다
- 예: `안양 석고보드` → `안양에서 석고보드를 찾는 경우`
- `title`, `meta_description`, `tags`는 원문 사용 가능
- 본문에서는 자연어 변형 우선
- SEO 키워드를 고유명사처럼 반복하지 않는다

## 자동 검수 규칙

`fact_safety_check`

- `contains_exact_price`: 본문에 근거 없는 원 단위 가격이 포함되면 `true`
- `contains_unsourced_percent`: 제공 자료 없는 % 수치가 있으면 `true`
- `contains_unsourced_lifespan`: 제공 자료 없는 수명/내구연한 수치가 있으면 `true`
- `contains_awkward_seo_phrase`: 지역명+자재명 키워드를 어색하게 반복하면 `true`
- `comparison_sections_use_tables`: 자재/성능/용도 비교 섹션이 Markdown 테이블을 사용하면 `true`
