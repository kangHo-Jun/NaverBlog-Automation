# 07_디버깅가이드

## 우선 확인 순서
1. 시트1 A~D, F, J 값 확인
2. 입력 폴더에 새 파일이 실제로 있는지 확인
3. JSON 출력 폴더에 `_preprocess.json` 생성 여부 확인
4. `_final_seo.json` 생성 여부 확인
5. 결과 출력 폴더에 `.html` 저장 여부 확인

## 자주 발생하는 오류

### 입력 파일이 처리되지 않음
- 원인:
  - 입력 폴더가 비어 있음
  - 이미 처리된 파일로 인식됨
  - 파일 형식이 기대와 다름
- 확인:
  - `getNewFilesToProcess()`
  - `isAlreadyProcessed()`

### 스타일/템플릿을 못 읽음
- 원인:
  - 시트2/시트3 누락
  - B2 또는 D2 값 오류
- 확인:
  - `getStyleDataFromSheet()`
  - `getSelectedTemplate()`

### Gemini 분석 실패
- 원인:
  - `GEMINI_API_KEY` 미설정
  - 응답 JSON 파싱 실패
- 확인:
  - `STEP_B_geminiAnalysis()`

### Claude HTML 생성 실패
- 원인:
  - `CLAUDE_API_KEY` 미설정
  - 프롬프트 과대
  - API 응답 오류
- 확인:
  - `processNextSEOFile_V7HTML()`
  - `createV7HTMLPrompt()`

### 최종 HTML 저장 실패
- 원인:
  - `_final_seo.json` 없음
  - `content` 필드 비어 있음
  - 결과 폴더 권한/설정 문제
- 확인:
  - `STEP_D2_SaveAsHTML()`

### 이미지 관련 오류
- 원인:
  - `image1_url` 비어 있음
  - 플레이스홀더 수와 이미지 수 불일치
  - 직접업로드 파일명이 숫자로 시작하지 않음
- 확인:
  - `processAutoImage_()`
  - `processManualImage_()`
  - `getManualImageFileSummary_()`

## 상태값 해석
- `글생성중`
- `발행준비`
- `이미지생성중`
- `이미지생성완료`
- `오류`
