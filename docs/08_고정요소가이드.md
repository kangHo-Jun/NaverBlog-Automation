# 08_고정요소가이드

## 개요
- V7 HTML은 고정 표준 템플릿을 Claude에게 직접 생성시키는 구조다
- 따라서 CTA, FAQ, 서명블록은 후처리 함수가 아니라 `createV7HTMLPrompt()`의 지시문으로 고정된다

## 고정 요소 3종

### CTA박스
- 목적: 상담/문의 유도
- 고정 링크: `https://daesan.ai`
- 필수 위치:
  - 모든 섹션 본문 종료 후
  - FAQ 바로 앞
- 금지 위치:
  - 문서 상단
  - 첫 섹션 이전

### FAQ 스키마
- 목적: 검색 친화형 FAQ 구조
- 형식:
  - `FAQPage`
  - `Question`
  - `acceptedAnswer`
- 필수 위치:
  - CTA박스 다음
- 원칙:
  - 6개 작성
  - `Q/A` 접두사 없음

### 서명블록
- 목적: 글 출처와 브랜드 신뢰 보강
- 내용:
  - `(주)대산 기술팀`
  - `30년 건축자재 전문 노하우`
  - `daesan.ai`
- 필수 위치:
  - 마무리 다음
  - 문서 맨 마지막

## 현재 고정 HTML 규칙

### CTA박스
```html
<div style="background:#1a3a5c;color:white;padding:2em;margin:2em 0;border-radius:8px;text-align:center;">...</div>
```

### FAQ 스키마
```html
<div itemscope itemtype="https://schema.org/FAQPage">...</div>
```

### 서명블록
```html
<div style="border-top:1px solid #ddd;margin-top:3em;padding-top:1.5em;color:#666;font-size:0.9em;line-height:1.8;">...</div>
```

## 운영 주의점
- 위치 지시가 약하면 모델이 CTA를 상단에 배치할 수 있다
- 따라서 문구보다 위치 규칙을 먼저 강제해야 한다
- 변경은 가능하지만 반드시 `createV7HTMLPrompt()`와 함께 맞춰야 한다
