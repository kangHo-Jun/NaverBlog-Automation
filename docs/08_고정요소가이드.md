# 08_고정요소가이드

## 개요
모든 블로그 글에 자동으로 삽입되는 고정 요소 목록과 관리 방법.
고정 요소는 `convertToBloggerHTML()` 내부에서 처리된다.

---

## 고정 요소 목록

| 요소 | 함수 | 위치 |
|------|------|------|
| CTA (견적받기) | `buildCtaHtml()` | 본문 중간 + 끝 |
| 회사 서명 | `buildSignatureHtml()` | 글 최하단 |
| 관련 글 | `buildRelatedPostsHtml()` | 글 하단 |
| FAQ | `buildFaqHtml()` | 글 하단 |

---

## 1. CTA (견적받기)

### 삽입 위치
- 본문 중간: H2 섹션 기준 중간 지점 자동 계산
- 본문 끝: 글 마지막

### 현재 설정
- 버튼 링크: `https://daesan.ai`
- 버튼 문구: `견적 받기 →`
- 배경색: 다크 네이비

### 수정 방법
- `buildCtaHtml(materialName)` 함수 수정
- `materialName` 은 본문 첫 번째 SEO 키워드 자동 사용

---

## 2. 회사 서명 (Signature)

### 삽입 위치
- 글 최하단 고정

### 현재 설정
- 공식 사이트: `daesan.ai`
- 링크: `https://daesan.ai`

### 수정 방법
- `buildSignatureHtml()` 함수 수정

---

## 3. 관련 글 (Related Posts)

### 삽입 위치
- 글 하단 CTA 이후

### 현재 설정
- `buildRelatedPostsHtml(posts)` 로 생성
- posts 데이터는 `final_seo.json` 기반

### 수정 방법
- `buildRelatedPostsHtml()` 함수 수정

---

## 4. FAQ

### 삽입 위치
- 글 하단

### 현재 설정
- `buildFaqHtml(faqItems)` 로 생성
- faqItems 는 Claude 본문 생성 시 자동 포함

### 수정 방법
- `buildFaqHtml()` 함수 수정

---

## 고정 요소 수정 시 주의사항

- 고정 요소는 모든 글에 동일하게 적용된다
- 링크/문구 변경 시 `convertToBloggerHTML()` 내 해당 함수만 수정
- 디자인 변경 시 네이비 `#2c5f8a` 색상 통일 유지
- 기존 원본 HTML의 `<footer>` 는 자동 제거됨
