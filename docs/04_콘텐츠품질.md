# 04_콘텐츠품질

## 핵심 원칙
- 근거 없는 수치 생성 금지
- 과장 광고 문구 금지
- 대산 브랜드는 유통/공급 포지션으로만 자연스럽게 노출
- AI 티 나는 상투 문장을 줄이고, 현장 설명처럼 읽히게 구성
- 자재별 시공 주의사항은 별도 Markdown 파일에서 동적 주입

## 브랜드 노출 원칙
- 허용 표현
  - `대산을 통해 공급되는`
  - `대산에서 취급하는`
  - `대산 공식 유통`
- 글 1편당 1~2회만 자연스럽게 노출
- 브랜드를 주인공처럼 과잉 반복하지 않는다
- 제품 언급 시 공급처 문맥 연결은 허용
  - 예: `LX PF보드 (대산 공급)`

## AI 티 방지 원칙
- `~를 통해` 남발 금지
- `~에 대해` 남발 금지
- `~되어진다` 같은 이중피동 금지
- `결론적으로`, `따라서`, `이를 통해` 반복 제한
- `또한`, `나아가`, `아울러` 문두 반복 제한
- `시사하는 바가 크다` 같은 전형적 AI 문구 금지
- 연결어미 뒤 쉼표 금지
- 문장 길이 균일화 금지

## 수치/사례 품질 기준
- 제공 자료 없는 가격 생성 금지
- 출처 없는 퍼센트 생성 금지
- 출처 없는 수명 단정 금지
- 법규/인증은 참고 기준으로만 설명

## 문체 품질 기준
- 도입은 독자 공감형 인삿말 100자 이상
- 회사 시점 유지
- 상단에 CTA, 서명블록 배치 금지
- 과장된 판매 문구보다 판단 근거 우선
