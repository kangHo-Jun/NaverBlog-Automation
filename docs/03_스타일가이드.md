# 03_스타일가이드

## 개요
- 시트2는 글의 문체와 문장 결을 조절하는 기준표다
- 시트1 `B2`의 숫자를 읽어 해당 스타일 행을 선택한다
- 핵심 함수:
  - `initializeStyleSheet()`
  - `getStyleDataFromSheet()`
  - `generateStyleDescription()`

## 시트2 구조
- A열: 번호
- B열: `writing_tone`
- C열: `sentence_style`
- D열: `personal_touch`
- E열: `visual_richness`
- F열: `narrative_flow`
- G열: `expertise_level`
- H열: `avg_sentence_len`
- I열: `style_features`

## 각 항목 의미
- `writing_tone`
  - 전체 글의 친근함/전문성 톤
- `sentence_style`
  - 문장 형식의 단정함, 대화감, 설명감
- `personal_touch`
  - 너무 건조하지 않게 만드는 인간적 결
- `visual_richness`
  - 표, 강조, 시각 포인트 밀도 기준
- `narrative_flow`
  - 도입에서 본문, 정리까지 연결감
- `expertise_level`
  - 현장성, 기술성, 판단 근거의 밀도
- `avg_sentence_len`
  - 평균 문장 길이 기준
- `style_features`
  - 자유 메모 성격의 보조 특징

## 운영 방식
1. 시트2에서 스타일 레코드를 준비
2. 시트1 `B2`에 번호 입력
3. 실행 시 `getStyleDataFromSheet()`가 해당 레코드 로드
4. `createV7HTMLPrompt()` 내부 `styleInstructions`로 반영

## 기본값 처리
- B2가 비어 있거나 숫자가 아니면 `1` 사용
- 번호를 찾지 못하면 코드 내 기본 스타일 객체 사용

## 실무 권장값
- 극단값보다 5~8 범위 중심 운영이 안정적
- `avg_sentence_len`은 35~55자 정도가 무난
- `style_features`에는 금지어 목록보다 방향 메모를 쓰는 편이 좋다
