🏗️ 네이버 에디터 전용 건축자재 블로그 작성 지침서 v1.0
===============================================================================

📋 개요
이 지침서는 네이버 블로그 SmartEditor ONE에 최적화된 건축자재 블로그 글 작성 가이드입니다.
HTML/CSS 없이 네이버 에디터 네이티브 기능만으로 전문적인 블로그를 작성합니다.

===============================================================================
[1] 기본 원칙
===============================================================================

1. 출력 형식
   - 마크다운이나 HTML이 아닌 **서식 있는 텍스트(Rich Text)** 형태로 작성
   - Google Docs 또는 Word에서 작성 후 → 네이버 에디터에 복사 붙여넣기
   - 복사 후 네이버 에디터에서 인용구/표 스타일/구분선 추가 적용

2. 어조 및 스타일
   - 전문적이면서 친근한 톤 ("~합니다", "~드려요" 혼용)
   - 건축자재 비전문가도 이해할 수 있도록 쉬운 용어 사용
   - 1~2문장마다 줄바꿈으로 가독성 확보

3. 글 구조 (6~8개 섹션)
   - 도입 인삿말 (100자 이상)
   - 목차 (수동 작성)
   - 본문 3~5개 섹션 (각 200자 이상)
   - FAQ 3~6개
   - 마무리 및 CTA
   - 태그

[2] 네이버 에디터 서식 적용 가이드
===============================================================================

┌─────────────────────────────────────────────────────────────────────────────┐
│ 원하는 효과          │ 네이버 에디터 적용 방법                              │
├─────────────────────────────────────────────────────────────────────────────┤
│ 제목 강조            │ [문단 스타일] → "주제목" 또는 "소제목" 선택          │
│ 컬러 강조 박스       │ [인용구] → "포스트잇" 또는 "프레임" 스타일           │
│ 비교 테이블          │ [표] 삽입 → [표 스타일] 프리셋 선택                  │
│ 섹션 구분            │ [구분선] 삽입 (다양한 스타일 중 선택)                │
│ 핵심 강조            │ 텍스트 선택 → [배경색(형광펜)] 적용                  │
│ 목차                 │ 볼드체 + 구분선으로 수동 작성 (앵커 링크 불가)       │
│ FAQ                  │ Q를 볼드체로, A를 일반체로 구분                      │
│ 이미지 캡션          │ 이미지 삽입 후 하단 캡션 기능 사용                   │
└─────────────────────────────────────────────────────────────────────────────┘

===============================================================================
[3] 📸 사진 플레이스홀더 가이드 (건축자재 특화)
===============================================================================

글 작성 시 아래 형태로 사진 위치와 설명을 표시합니다.
작성 완료 후 네이버 에디터에서 실제 이미지로 교체합니다.

▶ 기본 사진 유형 (템플릿별 선택)

[제품소개형]
📷 사진 1: 제품 전체 외관 및 패키징
📷 사진 2: 제품 단면 구조 상세 컷
📷 사진 3: 일반 제품과 나란히 비교 모습
📷 사진 4: 실제 시공 현장 적용 사례
📷 사진 5: 시공 전후 비교 사진

[비교분석형]
📷 사진 1: 제품 A vs B 외관 나란히 비교
📷 사진 2: 성능 테스트 결과 (그래프/수치)
📷 사진 3: 내구성 테스트 모습
📷 사진 4: 가격대별 제품 라인업
📷 사진 5: 전문가 인터뷰 또는 인증서

[시공가이드형]
📷 사진 1: 필요한 도구 및 준비물 전체 샷
📷 사진 2: 작업 전 현장 상태
📷 사진 3: 단계별 시공 과정 (3~5장)
📷 사진 4: 주의해야 할 포인트 클로즈업
📷 사진 5: 완성된 시공 결과물

[트러블슈팅형]
📷 사진 1: 문제가 발생한 상황 (손상/결함)
📷 사진 2: 원인 분석 클로즈업
📷 사진 3: 해결 과정 단계별
📷 사진 4: 수리/교체 완료 후 모습
📷 사진 5: 예방을 위한 팁 시각화

[트렌드리뷰형]
📷 사진 1: 최신 트렌드 적용 사례
📷 사진 2: 국내외 우수 시공 사례
📷 사진 3: 디자인 컬러/패턴 옵션
📷 사진 4: 전시장/쇼룸 현장
📷 사진 5: 인테리어 조화 예시

▶ 사진 배치 원칙
- 전체 글에 **최소 8~12장** 사진 플레이스홀더 포함
- 각 섹션마다 **최소 1~2장** 배치
- 비교표 직전/직후에 비교 사진 배치 권장
- 복잡한 개념 설명 시 다이어그램/인포그래픽 권장

▶ 플레이스홀더 작성 형식
   📷 [사진 N: 구체적인 설명]
   
   예시:
   📷 [사진 3: 친환경 석고보드 단면 구조 - 다공성 흡습층 강조]
   📷 [사진 7: 일반 석고보드(왼쪽) vs 친환경 석고보드(오른쪽) 나란히 비교]

[4] 글 구성 템플릿 (5단계)
===============================================================================

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[STEP 1] 제목 + 인삿말 + 목차

■ 제목 (25~40자)
- SEO 키워드 포함
- 숫자 활용 권장 (예: "2025년 꼭 알아야 할 ~", "BEST 5")
- 클릭 유도하되 과장 금지

■ 부제목 (한 줄)
- 제목을 보완하는 설명

■ 인삿말 (100자 이상)
- 독자의 고민 공감 ("이런 적 있으시죠?")
- 글에서 얻을 수 있는 가치 제시
- 친근한 톤

■ 목차 (6개 항목)

4. [섹션4 제목] 🏅

[STEP 2] 본문 섹션 1~3

■ 각 섹션 구성
- 소제목 (이모지 포함)
- 본문 (200자 이상)
- 비교표 (해당 시 - 네이버 표 기능 사용)
- 사진 플레이스홀더 1~2개
- 💡팁 또는 ⚠️주의사항 인용구 박스

■ 섹션 2 비교표 예시 (네이버 표로 변환)
┌──────────────┬───────────────┬───────────────┐
│ 구분         │ 일반 석고보드  │ 친환경 석고보드│
├──────────────┼───────────────┼───────────────┤
│ 포름알데히드 │ 0.3~0.5 mg/L  │ 0.1 mg/L 이하 │
│ 습도 조절    │ 제한적        │ 우수          │
│ 가격 (3.3㎡) │ 8,000~12,000원│ 15,000~25,000원│
└──────────────┴───────────────┴───────────────┘

[STEP 3] 본문 섹션 4~6

■ 동일한 구성 반복
- 각 섹션별 사진 플레이스홀더 포함
- 핵심 정보는 인용구(포스트잇) 활용

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[STEP 4] FAQ (자주 묻는 질문) 3~6개

■ 형식 (Q&A 앞에 Q, A 붙이지 않음)

❓ 친환경 석고보드는 정말 효과가 있나요?

네, 한국건설기술연구원 실험 결과 포름알데히드 농도가 약 70% 낮게 측정되었습니다.

❓ 가격 차이가 많이 나는데 꼭 써야 할까요?

장기적으로 건강 비용 절감과 집 자산가치 상승을 고려하면 충분히 가치 있는 투자입니다.

❓ 기존 집에도 교체할 수 있나요?

리모델링 시 기존 석고보드를 철거하고 교체 가능합니다. 30평 기준 1~2주 소요됩니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[STEP 5] 마무리 + 태그

■ 마무리 인삿말 (150자 이상)
- 글 요약
- 독자와 소통 유도 ("궁금한 점 댓글로 남겨주세요!")
- CTA (상담/견적 문의)

■ 태그 (10개, # 없이 쉼표 구분)
친환경석고보드, 석고보드, 건강한집, 새집증후군, 친환경건축자재, 
실내공기질, 리모델링, 인테리어, HB마크, 환경표지인증

[5] 워크플로우 요약
===============================================================================

1️⃣ AI(Claude/GPT)로 이 지침에 맞춰 글 생성 요청
   ↓
2️⃣ Google Docs에 붙여넣기 + 기본 서식 적용
   - 제목: "제목 1", "제목 2" 스타일
   - 볼드, 색상 강조
   - 기본 표 삽입
   ↓
3️⃣ 전체 복사 → 네이버 에디터에 붙여넣기
   ↓
4️⃣ 네이버 에디터에서 마무리 (핵심!)
   ✅ 💡팁/⚠️주의 박스 → [인용구] → 포스트잇 스타일
   ✅ 비교표 → [표 스타일] 프리셋 적용
   ✅ 섹션 사이 → [구분선] 삽입
   ✅ 📷 플레이스홀더 → 실제 이미지로 교체 + 캡션 추가
   ↓
5️⃣ 발행 전 최종 확인
   - 모바일 미리보기 체크
   - 이미지 로딩 확인
   - 태그 입력
