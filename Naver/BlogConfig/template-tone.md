# V7 HTML 템플릿 — 문체 & 작성 원칙

Apps Script의 `loadTemplateConfig_()` 가 `## PROMPT_TEXT` 이하를 추출해 시스템 프롬프트에 주입합니다.
이 파일을 Google Drive에 업로드하면 Code.gs 수정 없이 역할/원칙을 바꿀 수 있습니다.

## 수정 가이드

- **회사 시점 문구 변경**: `## PROMPT_TEXT` 의 `[시점 원칙]` 섹션 직접 편집
- **금지 패턴 추가**: `[수치/사례 원칙]` 아래 줄 추가
- **CTA 기본 원칙 변경**: `[작성 원칙]` 내 항목 편집
- 파일 저장 후 반드시 Google Drive에 재업로드 필요
- 변경사항은 CacheService 만료(6시간) 후 또는 `clearTemplateCache()` 실행 후 반영
